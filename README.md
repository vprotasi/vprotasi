# Welcome to my homepage!

## Here you will find:

### `wave-pde-simulation`

This is a project I completed during the autumn semester of 2023 at the University of
Tyumen. I got inspired to make this string visualiser at a lecture on partial
differential equations (also known as mathematical physics). This visualiser can
produce the string oscillations that result after one of the three different
types of initial stimulus: pull, point strike, and hammer strike with a hammer
whose head diameter is non-negligible.
